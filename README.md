## 🌱 보호자 연계형 멀티모달 AI 스마트 화분 시스템

고령자의 정서 상태와 일상 활동을 자연스럽게 모니터링하고, 보호자와의 정서적 연결을 지원하는 AI 기반 스마트 화분 시스템입니다.

본 프로젝트는 독거노인의 정서적 고립, 우울감, 고독사 문제를 해결하기 위해 기획되었으며, 
화분이라는 친숙한 오브젝트를 인터페이스로 활용하여 거부감 없이 일상 속에서 지속적인 상호작용이 가능하도록 설계되었습니다.

---

## 🎯 프로젝트 목적

- 고령자의 **정서 상태를 비접촉 방식으로 지속 모니터링**
- 일상 행동(물 주기 등)을 기반으로 **생활 패턴 간접 분석**
- 보호자에게 상태를 전달하여 **원격 돌봄 환경 구축**
- 식물 성장 경험을 통한 **정서적 안정 및 성취감 제공**

---

## 🧠 시스템 구성

### 1. 센서 기반 데이터 수집 (ESP32)
- 토양 수분, 조도, 공기질, 시간 데이터 수집
- UART 통신을 통해 라즈베리파이로 전송

### 2. 정서 인식 및 사용자 인터랙션 (Raspberry Pi)
- CNN 기반 표정 인식 모델 (전이학습 + 양자화 적용)
- YOLO 기반 객체 인식 및 상황 분석
- Gemini API 기반 자연어 생성
- TTS를 통한 음성 피드백 제공

### 3. 식물 인식 및 성장 관리
- YOLOv8 기반 식물 성장 상태 인식
- 생성형 AI를 활용한 성장일지 자동 생성
- 조도 기반 자동 회전 화분 시스템 구현

### 4. 보호자 연계 시스템
- Pushbullet API를 통한 알림 전송
- 장기 미활동 및 우울 상태 감지 시 보호자 알림

---

## 🔄 동작 흐름

1. 사용자가 물을 주면 수분 센서가 이를 감지
2. ESP32가 환경 데이터를 수집하여 Raspberry Pi로 전송
3. 표정 인식 및 환경 분석 후 음성 피드백 제공
4. 식물 인식 시스템이 성장 상태 분석 및 성장일지 생성
5. 결과를 TTS로 사용자에게 전달
6. 주요 정보는 보호자에게 알림 전송
7. 조도 기반 자동 화분 회전 수행

---

## 📊 주요 성과

- 표정 인식 정확도 약 **80% 달성**
- 전체 처리 시간 **5초 이내 실시간 처리**
- ESP32-Raspberry Pi 간 **안정적인 데이터 통신 구현**
- 멀티모달 시스템 통합 후 **장시간 안정 동작 확인**
- 2025년 한국정보기술학회 하계종합학술대회의 대학생 논문경진대회 **동상 수상**
- 2025년 한국정보기술학회 추계종합학술대회의 대학생 논문경진대회 **동상 수상**
- 2025년 경북대학교 캡스톤디자인 **우수상 수상**

## 🛠️ 기술 스택

- Embedded: ESP32, Raspberry Pi 5 
- AI: YOLOv8, CNN (FER2013, RAF-DB), MobileNetV2
- Optimization: ONNX, FP16 Quantization
- Communication: UART, TCP Socket
- API: Gemini API, Pushbullet API
- TTS: gTTS
- Sensor: YL-83, BH1750, SGP30, DS3231
